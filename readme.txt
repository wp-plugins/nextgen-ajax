== Description ==

nextGEN_ajax is a plugin that adds the Ajax functionality to the nextGEN gallery. This way it’s not the complete page that will be reloaded but only the gallery part!
 
The version 0.1 also comes with the Slimbox (LightBox clone based on Mootools).

You can look for an example on http://www.dailyportal.de !
 
Further features will follow within the next weeks.

GERMAN:
________________________________________________________________
nextGEN_ajax ist ein Plugin für die nextGEN Gallery und erweitert sie um Ajax funktionalität. Somit wird nichtmehr die komplette Seite neu geladen sondern nur noch der Galerieteil!

In der Version 0.1 wird noch die Slimbox (LightBox Klon der auf den Mootools basiert) mitgeliefert.

Eine Demonstration kannst du dir unter http://www.dailyportal.de anschauen!

Weitere Eigenschaften folgen die nächsten Wochen.


== Installation ==

1. Upload `nextgen_ajax` to the `/wp-content/plugins/` directory
2. Aktiviere das Plugin unter 'Plugins' im WordPress Administrations MenŸ
